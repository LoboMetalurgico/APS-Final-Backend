# APS Final - EcoMobile

This repository contains the backend services for the APS Final project, code-named "EcoMobile".

This project consists of four main services:
1. **Gateway Service**: Acts as the main entry point for client requests, routing them to the appropriate microservices.
2. **Weather Service**: Provides weather data based on geographical coordinates.
3. **Geolocation Service**: Converts location names into geographical coordinates.
